## Component Breakdown

### Hero Section
- **Height:** 500px (400px on mobile)
- **Background:** Gradient purple-blue with image overlay
- **Content:** 
  - Title (3.5rem)
  - Subtitle (1.5rem)
  - Search input + button
- **Effect:** Semi-transparent dark overlay for text readability

### Destination Card
- **Dimensions:** Flexible width, ~400px height
- **Components:**
  - Image (220px height) with category badge
  - Title (1.4rem)
  - Location with icon
  - Description (truncated to 100 chars)
  - Budget info
  - View Details button
- **Effects:** 
  - Hover: translateY(-5px), enhanced shadow
  - Image zoom on hover

### Hotel Card
- **Dimensions:** Flexible width, ~350px height
- **Components:**
  - Image (200px height) with star rating
  - Hotel name
  - Destination name
  - Price per night
- **Effects:** Similar hover effects as destination card

## Color Usage

### Backgrounds:
- Hero: `linear-gradient(135deg, #667eea 0%, #764ba2 100%)`
- Cards: `#ffffff` (white)
- Page: `#f5f5f5` (light gray)
- Sections: `#f9f9f9` (very light gray)

### Text:
- Primary: `#333` (dark gray)
- Secondary: `#666` (medium gray)
- Muted: `#888` (light gray)

### Buttons:
- Primary: `#667eea` (purple-blue)
- Accent: `#ff6b6b` (coral red)
- Hover states: Slightly darker shades

### Badges:
- Category: `#e3f2fd` background, `#1976d2` text
- Budget: `#f3e5f5` background, `#7b1fa2` text
- Objective: `#667eea` background, white text

## Responsive Breakpoints

### Mobile (< 768px):
- Single column layouts
- Smaller fonts (title: 2rem, subtitle: 1.1rem)
- Stack form fields vertically
- Full-width buttons
- Reduced padding/margins
- Smaller image heights

### Tablet (768px - 1024px):
- 2-column grids for cards
- Moderate font sizes
- Flexible layouts

### Desktop (> 1024px):
- 3-column grids for cards
- Full font sizes
- Sidebar + main content layout
- Maximum content width: 1200px

## Animation Effects

### Transitions:
- All interactive elements: `0.3s ease`
- Hover effects: smooth scale/translate
- Button hover: `translateY(-2px)` or `translateY(-3px)`

### Shadows:
- Default: `0 2px 8px rgba(0, 0, 0, 0.1)`
- Hover: `0 8px 20px rgba(0, 0, 0, 0.15)`
- Button hover: `0 5px 15px rgba(color, 0.3)`

## Typography

### Font Sizes:
- Hero Title: 3.5rem (2rem mobile)
- Section Title: 2.5rem (1.8rem mobile)
- Card Title: 1.4rem (1.2rem mobile)
- Body: 1rem
- Small: 0.9rem
- Tiny: 0.8rem

### Font Weights:
- Bold: 700 (titles, buttons)
- Semi-bold: 600 (labels, subtitles)
- Normal: 400 (body text)

## Spacing System

### Padding:
- Page: 2rem (1rem mobile)
- Cards: 1.5rem
- Sections: 3rem margin-bottom

### Gaps:
- Grid gaps: 2rem (1.5rem mobile)
- Form gaps: 1.5rem
- Button gaps: 1rem
